# Cube internal-axis protocol — quarantined research track

Status: [HEURISTIC] programme design; no claim in the sealed paper is modified.

## Deflation first

A nonzero loop product on analyst-chosen couplings is a true invariant of a constructed object, not evidence about nature. Physical relevance requires a named device whose couplings are measured or fixed by its dynamics.

## Object typing

- Object A: directed coupling matrix on an N x N x N nearest-neighbour graph.
- Object B: complete transfer/resolvent matrix derived from A.
- Object C: scalar projection of B.
- Object D: a physical two-dimensional complex field, only if a device selects it.

R and Delta may be evaluated only on an explicitly typed complete kernel or operator. The SPARC/eml★ conjunction is inapplicable to A until Object D exists.

## Orthogonal axes

1. surface vs volume;

5. reciprocal vs directed couplings;
6. equal moduli vs disparate moduli vs depth-correlated moduli;

9. chosen couplings vs couplings derived from a named physical generator;
10. finite graph vs physically justified mode/size limit.

## Required raw outputs

Per instance and per cycle: forward product, reverse product, both phases, both log-moduli, asymmetry ratio, smallest reverse-link modulus, quasi-pole flag, and changes under each transformation class. Never print a physical verdict automatically.

## Positive controls

- reciprocal network: forward and reverse data coincide under the declared convention;
- exact gradient GL(1,C) site transformation: directed cycle products telescope exactly;
- Landau/Peierls plaquette with analytically known flux;
- open-chain 2-cycle with prescribed product.

## Negative sweep

- omit one edge;
- reverse one edge;
- apply a non-gradient linkwise mutation;
- permute couplings while keeping geometry fixed;
- evaluate R on a scalar projection instead of the complete matrix;
- force an artificial complex plane or synthetic scale.

## eml★ gate

No eml★ candidate is admitted unless a named physical system supplies a single 2D carrier and a field f(z,zbar) that passes every clause of the project conjunction. A finite cube resolvent is rational; transcendence must come from an independently justified physical limit, not from encoding.
